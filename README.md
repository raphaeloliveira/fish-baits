# fish-baits
List of functions for [Fish Shell](https://fishshell.com/).

## Commands

- `g` = `git`
- `gs` = `git status`
- `gfp` = `git fetch -p`
- `gba` = `git branch -a`
- `gmof` = `git merge origin/$current_branch --ff-only`

##Installation
Copy the `*.fish` files into `~/.config/fish/functions/`.
